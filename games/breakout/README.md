# Breakout

Breakout will be the second game in Tiny Arcade Agent.

This game will build on the ideas from Pong and introduce the first
reinforcement learning concepts. The player controls a paddle, the ball breaks
bricks, and the game can be described as an environment where an agent takes
actions and receives rewards.

## Planned learning goals

Students will use Breakout to explore:

- A richer game state with paddle, ball, bricks, score, and lives.
- Actions such as move left, move right, or stay.
- Rewards for useful events, such as breaking a brick.
- Episodes that start, continue, and end.
- The difference between a hand-written controller and a learning agent.
- The basic reinforcement learning loop: observe, act, receive reward, update.

## Status

Breakout is planned but not implemented yet. For now, start with
[Pong](../pong/README.md) to learn the project structure and the basic game loop.
